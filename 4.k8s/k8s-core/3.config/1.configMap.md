## ğŸ“¦ **Using ConfigMaps in Pods**

You can inject ConfigMaps into Pods in **three main ways**.

---

### **1ï¸âƒ£ As Environment Variables**


### **2ï¸âƒ£ As Individual Env Vars (Selective Import)**

```yaml
        name: app-config
```

ğŸ§  Use this when you only need _specific keys_.

---

### **3ï¸âƒ£ As Mounted Files (Volumes)**

```yaml

  name: file-demo
spec:
  containers:
    - name: app
      image: alpine
      command: ["sh", "-c", "cat /etc/config/APP_MODE; sleep 3600"]

        - name: config-volume

    - name: config-volume

        name: app-config
```

Now inside the container:

```bash

```

Output:


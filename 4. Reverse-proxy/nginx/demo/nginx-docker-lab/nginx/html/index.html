Docker Compose + Nginx playground

Nginx learning lab

This project is a hands-on sandbox for reverse proxy, load balancing, TLS, static files, caching, rate limiting, rewrites, custom errors, and basic auth.

Test load balancing Test caching Test basic auth

Main routes

  • /app1/ -> direct proxy to backend 1
  • /app2/ -> direct proxy to backend 2
  • /api/ -> upstream with load balancing
  • /cache/ -> proxy cache enabled
  • /rate/ -> rate limiting enabled

Infrastructure routes

  • /healthz -> Nginx health endpoint
  • /status -> Nginx stub status
  • /old-docs -> redirect example
  • /rewrite-demo/headers -> rewrite example
  • HTTPS on port 8443

### How to enable

helm values:

```yaml
routingMode: "native"

ipv4NativeRoutingCIDR: "10.244.0.0/16"
ipv6NativeRoutingCIDR: "5be8:dde9:7f0b:d5a7:bd01:b3be:9c69:573b/112"
```

# VIM Commands

## Insert Mode

- `i`: before cursor
- `a`: after cursor
- `o`: new line below

- `SHIFT + i`: beginning of line
- `SHIFT + a`: end of line
- `SHIFT + o`: new line above

---

## Movement

- `h`: left
- `j`: down
- `k`: up
- `l`: right

> Config
>
> - `:set number`, remove by `:set nonumber`
> - `:set relativenumber`, remove by `:set norelativenumber`
> - `:set cursorline` , remove by `:set nocursorline`
> - `:set cursorcolumn`, remove by `:set nocursorcolumn`

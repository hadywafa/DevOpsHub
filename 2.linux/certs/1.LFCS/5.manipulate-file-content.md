# Manipulate-file-content

## 1. Show content of file

### `cat` / `tac`

- cat : show content of file
- tac : show content of file in reverse

```bash
cat <file-name>

# examples
cat test1.txt
```

```bash
tac <file-name>

# examples
tac test1.txt
```

---

### `head` / `tail`

- head: show first n lines of file (default 10)
- tail: show last n lines of file (default 10)

```bash
head -n <line-numbers>

# examples
head -n 5 test1.txt
```

```bash
tail -n <line-numbers>

# examples
tail -n 5 test1.txt
```

---

## 2. Transform content of file

### `sed`

- sed: stream editor for filtering and transforming text files

```bash
sed -i <expression> <file-name>

# examples
sed -i 's/old/new/g' test1.txt
```

### `cut`

- cut: extract fields from text files

```bash
cut -f <field-numbers> <file-name>

# examples
cut -f 1 test1.txt
```

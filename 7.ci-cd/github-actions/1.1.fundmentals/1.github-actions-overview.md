### ğŸ§¬ **Workflow Structure**

A workflow is made of:

- **Event trigger** â†’ e.g., `on: push`
- **Jobs** â†’ e.g., `build`, `test`, `deploy`
- **Steps** â†’ individual commands or actions within a job
- **Runners** â†’ environments that execute the jobs

```yaml

  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Run tests
        run: npm test
```
